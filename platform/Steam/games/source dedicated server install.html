Source Dedicated Server (Выделенный сервер для Source)

"Source Dedicated Server" - это программа, позволяющая создать сервер для игры через Steam по локальной сети или в сети Интернет. (Если Вы просто хотите поиграть по сети, то Вам не нужен Source Dedicated Server - Вы можете вступить в сетевую игру, выбрав пункт "Серверы" в меню Steam.)

Если Вы хотите запустить игровой сервер, на который игроки смогут зайти через сетевое подключение, то самый простой способ это сделать - нажать кнопку "Загрузить и установить", расположенную ниже. Dedicated Server будет добавлен в список "Инструменты" и станет доступен уже через пару минут.

Продвинутые пользователи могут запустить Dedicated Server под операционной системой Linux. Щёлкните сюда, чтобы загрузить версию для Linux.

Имейте в виду, что для запуска любой версии программы Dedicated Server может понадобиться некоторая настройка сети (открытие портов и т.п.).
