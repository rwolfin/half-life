Source Software Developer's Kit (Набор разработчика ПО для Source)

Source SDK - это набор инструментов и ресурсов, необходимых для создания своего контента, модифицирования существующих или создания абсолютно новых игр, использующих технологию движка Source. Скачать SDK и попробовать работать в нём может каждый, но не забывайте, пожалуйста, что он спроектирован под продвинутых пользователей.

Включает в себя:
  • Hammer
  • Half-Life Model Viewer
  • Face Poser
  • Scene Manager
  • Различные компиляторы контента
  • Документацию
  • Исходный код и примеры
Для получения помощи от сообщества и изучения сетевой базы знаний, пожалуйста, посетите Valve Editing Resource Center (VERC).

Время от времени SDK будет пополняться новыми ресурсами, исходным кодом или примерами. Как и в случае с играми в Steam, эти обновления будут получены клиентом Steam автоматически.
